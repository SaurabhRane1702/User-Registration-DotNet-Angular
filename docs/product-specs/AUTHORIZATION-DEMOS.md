# Authorization Demos

The repo includes dedicated pages and endpoints to demonstrate role- and policy-based access rules.

## Demos

- Admin only
- Admin or teacher
- Library members only
- Female teachers applying for maternity leave
- Females under 10

## Purpose

These flows are useful as examples when adding new protected features because they show the current pattern across:

- backend role/policy enforcement
- frontend route claim checks
- unauthorized UX handling
